Static page being server from wwwroot folder.

    For this we can use:
  • app.UseDefaultFiles();
  • app.UseStaticFiles();
  • Or instead of these two we can use: app.UseFileServer();
If we pass a url like /index.html or / then we will get static html file
else we will get the greeting message from config. Try it out!
